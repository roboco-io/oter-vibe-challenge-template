# PROGRESS — 데일리 진행 인덱스

이 문서는 데일리 push와 회고(`retros/`)를 한눈에 보기 위한 인덱스다. 매일 push하기
전(또는 후) 아래 표에 한 줄을 추가한다. Retrobot이 자동 생성하는 KPT 회고
(`retros/YYYY-MM-DD-HHMM.md`)는 표에서 링크만 걸고 내용을 복사하지 않는다.

## 진행 로그

| 날짜 | 요약 | 회고 링크 |
|---|---|---|
| YYYY-MM-DD | 그날 한 작업을 한 줄로(예: "레거시 앱 기동 확인, E2E baseline 확보") | [retros/YYYY-MM-DD-HHMM.md](../retros/YYYY-MM-DD-HHMM.md) |

## 참고

- "요약"은 커밋 로그를 그대로 옮기지 말고, 그날의 의사결정/막힌 지점/해결 여부를
  한 줄로 압축한다. 평가자는 이 표를 먼저 훑고 필요하면 회고 원문으로 들어간다.
- 회고 링크가 여러 개인 날(예: 하루에 두 번 push)은 표 행을 추가한다.
- 회고 파일이 아직 없다면(예: 당일 push 전) 링크 칸을 `(pending)`으로 남겨 두고,
  push 이후 Retrobot이 파일을 생성하면 갱신한다.
